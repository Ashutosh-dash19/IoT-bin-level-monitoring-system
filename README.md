# IoT-bin-level-monitoring-system

This project uses Arduino and sensors to monitor the fill level of a bin and send data to ThingSpeak.

## Components Used
- Arduino
- Ultrasonic Sensor
- ESP8266 Module
- OLED Display

## Files
- `Arduino.ino` → main Arduino code
- `Circuit Connection.jpg` → Circuit Connection image
- `Console Output.png` → Console output image
- `Thingspeak Output.png` → Thingspeak output image


## Output
Screenshots of serial monitor / ThingSpeak graphs are also included.
